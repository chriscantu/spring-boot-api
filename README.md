Spring-Boot-API
===============

## Dependencies
* Java 1.8 or Higher
* Gradle 2.10

## Project Setup
* `git clone git@github.com:chriscantu/spring-boot-api.git`
* `cd spring-boot-api`
* `gradle bootRun`
* open `http://localhost:8080/hello`

## Testing
* `gradle test` - Runs all unit tests
* `gradle check` - Run all tests

## Other
* `gradle tasks` - Get a list of built in gradle tasks
